## Current mathematical target

The target is no longer merely to prove componentwise truncated domination.  The sharper sufficient target is a signed linear surplus bound:

\[
(\sigma_{\rm rel}-S_{\rm rel})+(\sigma_{\rm nonrel}-S_{\rm nonrel})\le \theta t,
\qquad \theta<1.
\]

Equivalently, in Lean-style form:

```lean
def SignedSurplusLinearBound (θ : ℚ) : Prop :=
  ∀ n t : ℕ,
    (SignedRelevantSurplus n t : ℚ)
      + (SignedNonRelevantSurplus n t : ℚ) ≤ θ * t
```

If this is proved for any fixed `θ < 1`, then the exact identity

\[
2f(n,t)-t=S(n,t)-\sigma(n,t)
\]

implies

\[
f(n,t)\ge \frac{1-\theta}{2}t,
\]

which proves Erdős #461.

Item 5 is the major breakthrough target.
